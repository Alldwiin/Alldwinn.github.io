Serveur API Rest : Pizzaland

Pizzaland est une API Java permettant la gestion d’une base de données via une servlet et le modèle MVC utilisant DTO et DAO.

L’API permet d’afficher des listes de pizzas, d’ingrédients ou de commandes tout en format JSON. Il permet aussi de modifier des commandes ou des pizzas.

Ce projet était une SAE de BUT 2éme année de Web Backend faite par groupe de 2 personnes. Durant ce projet je me suis principalement occupé de documenter et faire l’accès à la base de données ainsi que du réfacto du code.

Durant ce projet, j’ai appris le fonctionnement du format JSON, ainsi que l’utilisation de l’application Bruno et de la création d’une API jusqu’à sa documentation.
